# The_Global_Coffee_Health_Dataset_Project
SI 385 - Data Exploration Fall 2025

This project explores the Global Coffee Health Dataset to understand how coffee consumption is connected to health and lifestyle variables. The analysis includes data cleaning, exploratory data analysis (EDA), and visualizations created with Python libraries such as pandas and matplotlib.

Tools & Technologies:
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook
